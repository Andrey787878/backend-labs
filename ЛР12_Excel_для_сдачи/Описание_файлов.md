# ЛР12: Excel-файлы для сдачи

Эти файлы можно загружать в Swagger в ручку `POST /api/attendance/calculate`.

| Файл | Что показать | Ожидаемый результат |
| --- | --- | --- |
| `01_успешный_расчет_зачета.xlsx` | Валидный файл: несколько групп, пустая подгруппа, авто-зачет, одинаковое ФИО в разных группах | `200 OK` |
| `02_ошибка_нет_листа_посещаемость.xlsx` | В Excel нет листа `Посещаемость` | `422` |
| `03_ошибка_нет_колонки_фио.xlsx` | Нет обязательной колонки `ФИО` | `422` |
| `04_ошибка_неверный_тип_занятия.xlsx` | В колонке `Тип занятия` указано `seminar` вместо `lect` или `lab` | `422` |
| `05_ошибка_неверная_дата.xlsx` | Дата указана в неверном формате | `422` |
| `06_ошибка_подгруппа_ноль.xlsx` | Подгруппа равна `0`, а должна быть положительным числом | `422` |
| `07_ошибка_битый_xlsx_не_excel.xlsx` | Файл имеет расширение `.xlsx`, но внутри не Excel | `422`, без `500` |

Как показывать на сдаче:

1. Запусти проект и открой Swagger.
2. Авторизуйся админом.
3. Открой раздел `attendance`.
4. В ручке `POST /api/attendance/calculate` нажми `Choose File` и выбери нужный файл из этой папки.
